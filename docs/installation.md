# Installation

## Stable release

To install nextdep_dsp, run this command in your terminal:

```sh
uv add nextdep_dsp
```

Or if you prefer to use `pip`:

```sh
pip install nextdep_dsp
```

## From source

The source files for nextdep_dsp can be downloaded from the [Github repo](https://github.com/wmorellato/nextdep_dsp).

You can either clone the public repository:

```sh
git clone https://github.com/wmorellato/nextdep_dsp
```

Or download the [tarball](https://github.com/wmorellato/nextdep_dsp/tarball/main):

```sh
curl -OJL https://github.com/wmorellato/nextdep_dsp/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd nextdep_dsp
uv sync
```
